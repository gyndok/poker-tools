What it does

It finds your latest session, lists every hand you played, and walks you through them one at a time with a straight-talking GTO read — preflop ranges, bet sizing, calls vs. barrels, and rivers, adjusted for tournament ICM and stack depth. Ask for stats and it reports your VPIP, PFR, 3-bet, c-bet, WTSD and more, then tells you your style (TAG, LAG, etc.) and where you're leaking. It names mistakes plainly — but separates real leaks from coolers.

What you need

  • A Claude account.
  • Claude Cowork (the desktop app) or Claude Code. The tool reads the hand-history files on your computer, so it needs one of these — it will not work in the plain website chat.
  • Your ACR/WPN hand histories saved on your computer (turn on hand-history saving in the poker client's settings).

You do not need to know GitHub, the command line, or how to write code. GitHub is just where the tool is stored — you never have to open it.

Install in 3 steps

  1. Open Claude Cowork (or Claude Code).
  2. Type these two lines to Claude, one at a time: /plugin marketplace add gyndok/poker-tools /plugin install poker-hand-review@poker-tools That's it — Claude downloads everything for you.
  3. When Claude asks, point it at your hand-history folder, then say: review my hands

Where are my hand histories?

In your poker client, make sure hand-history saving is turned on (usually under Settings). The client saves a folder of HH…​.txt files, one screen-name folder per player — for example …/AmericasCardroom/handHistory/<your_screen_name>/. When Claude asks for a folder, pick that one. Not sure where it is? Just tell Claude "help me find my America's Cardroom hand histories" and it'll look.

FAQ

Do I need a GitHub account?
No. The two install lines pull everything automatically. You never touch GitHub.
Do I need to code or use a terminal?
No. You type plain instructions to Claude like you would in any chat.
Does it work in the Claude website chat?
No — it needs to read files on your computer, so use the Cowork desktop app or Claude Code.
Which poker sites?
It's built for America's Cardroom / Winning Poker Network hand-history files, No-Limit Hold'em tournaments. Cash and Omaha are detected but get a lighter review.
Where do my hand histories go?
They stay on your computer and are read locally during the review. Nothing is posted publicly.
Is this a solver?
No — it's fast GTO-principle coaching. For exact frequencies, pair it with a real solver (GTO Wizard, PioSOLVER) — the tool will even frame the spot for you.
